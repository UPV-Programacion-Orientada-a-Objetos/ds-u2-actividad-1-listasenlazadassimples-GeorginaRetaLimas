\doxysubsection{Detailed Description}
Sensor que maneja presiones en formato enteros. 

DIFERENCIAS CON \doxylink{classSensorTemperatura}{Sensor\+Temperatura}\+:
\begin{DoxyItemize}
\item Usa \doxylink{classListaSensor}{Lista\+Sensor$<$int$>$} en lugar de \doxylink{classListaSensor}{Lista\+Sensor$<$float$>$}
\item El procesamiento es diferente, cuando promedia no elimina al chiquita
\end{DoxyItemize}

VENTAJA DEL POLIMORFISMO\+: Ambos sensores tienen la misma interfaz pero comportamientos distintos. El código cliente no necesita saber cuál es cuál
